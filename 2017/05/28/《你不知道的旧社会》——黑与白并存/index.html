《你不知道的旧社会》——黑与白并存

发表于 2017-05-28

应学妹推荐读了这本书，总体来说是本有意思的书。

作者林希在该书以杂文的形式介绍了传说中“万恶的旧社会”中天津的各式人和物。
虽说讲的是天津，不过倒也算是中国在那个年代的一个缩影吧。而我也算是跟着过了把旧社会生活的瘾。

旧社会是黑暗的，这是毋庸置疑的。那个时候最不值钱的是人命，最不敢作为的是政府，最吃亏的是底层劳动者。但旧社会也有它温暖的一面，绑匪有绑匪的法则，小绺有小绺的行规——土匪绑票第一不绑女性，第二不得撕票；小绺上路三不偷，一不偷状元府第，二不偷孤儿寡母，三不偷慈善人家。坏了行规，道上的人都不会放过你。

旧社会有骗人的行当，如看香、相士、烧痦子等；也有真正的手艺，如正骨术、架子工、拿瘊子等。

旧社会时，黑社会抢地盘，不打群架而是跳油锅，每方轮换着跳，直到一方没人跳为止决出胜负。船家打架，速战速决，没有时间讲理，谁对谁错不是打架的内容，打架就是见血，谁敢先拿一块砖头将自己的脑袋瓜砸破，谁就有理，不敢砸的自动认输。

旧社会是乱的，但规矩却是不能少的。黑道如何偷、杀、绑、乞，有黑道的规矩；白道如何欺、诈、谄、贿，有白道的规矩；就连普通人如何吃饭、如何拿箸、如何守老礼也都有普通人的规矩。

除此之外还有各式各样其他不为人知的有趣的事物就不一一列举了，有兴趣可以找来读一读。
